## ğŸš€ Getting Started

### Prerequisites
- Node.js 16+
- MongoDB Atlas account
- npm or yarn

### Installation

1. Clone the repository
```bash

git clone git@github.com:believeharsh/BlogHive.git

```

2. Install dependencies
```bash
npm install
```
3. Set up environment variables

4. Start development servers
```bash
# Frontend
npm run dev

# Backend
npm run dev
```
